# Frontend

Esta pasta contem as telas e scripts da interface.

Arquivos principais:

- `index.html`: login.
- `dashboard.html`: painel inicial.
- `pacientes.html`, `medicos.html`, `consultas.html`, `leitos.html`, `internacoes.html`, `prontuarios.html`, `log.html`: modulos do sistema.
- `css/style.css`: estilos globais.
- `js/api.js`: configuracao da API e helpers compartilhados.

Para abrir localmente, use `frontend/index.html`.
