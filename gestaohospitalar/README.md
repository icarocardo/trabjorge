# Gestao Hospitalar

Projeto separado em duas partes:

- `frontend/`: telas HTML, estilos CSS e scripts de interface.
- `backend/`: API Node.js do sistema.

## Como rodar

Backend:

```bash
cd backend
npm start
```

Frontend:

Abra `frontend/index.html` no navegador.

A API esperada pela interface continua configurada em `frontend/js/api.js` como `http://localhost:3000`.
